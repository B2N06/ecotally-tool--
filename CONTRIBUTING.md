# Contributing

Thanks for helping make ecological analysis more transparent.

1. Create a focused branch.
2. Add or update tests for every behavior change.
3. Run `python -m unittest discover -s tests -v`.
4. Update `CHANGELOG.md` when behavior visible to users changes.
5. Open a pull request explaining the ecological use case and assumptions.

Please keep the core dependency-light and make formulas auditable. Bug reports
should include a minimal anonymized input dataset and the expected result.
