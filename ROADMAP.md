# Roadmap

EcoTally evolves through small, tested releases. The next milestones are:

- richer temporal and repeated-measures summaries;
- optional spatial-distance and distance-decay analysis;
- confidence intervals for group effects;
- additional functional and taxonomic beta-diversity methods;
- performance profiling for very large community matrices;
- stable 1.0 schemas after public feedback on report and bundle formats.

Backward compatibility is best-effort before 1.0. Changes to formulas, report
keys, or bundle schemas must be documented in `CHANGELOG.md` and covered by
tests.
