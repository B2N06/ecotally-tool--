# Security policy

EcoTally processes local text files and does not make network requests, open
listening ports, or require a Windows Firewall exception. See
[Windows security and trust](WINDOWS_SECURITY.md) for release verification,
SmartScreen guidance, checksums, and the code-signing path.

Please report suspected vulnerabilities privately through GitHub's security
advisory feature rather than a public issue. Include the affected version, a
minimal reproducer, and the expected impact. Supported releases are the latest
minor version on the `main` branch.
